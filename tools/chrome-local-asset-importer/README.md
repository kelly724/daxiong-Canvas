# Infinite Canvas 本地素材导入 Chrome 插件

用于扫描当前网页图片，批量导入到素材库管理里的「本地素材」。

## 安装

1. 打开 Chrome：`chrome://extensions/`
2. 右上角开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本目录：`tools/chrome-local-asset-importer`

## 使用

1. 先启动 Infinite Canvas 后端服务。
2. 点击浏览器右上角插件图标。
3. 输入服务地址，例如 `127.0.0.1:8767`、`192.168.1.10:3000`，也可以填写完整地址 `http://192.168.1.10:3000`。
4. 点击「连接」。
5. 可点击图钉按钮打开 Chrome 侧边栏，侧边栏不会因为切换页面而关闭。
6. 在任意网页点击「扫描」，插件会列出当前页面可访问的图片。
7. 勾选图片，选择平台和模型。
8. 可开启「导入后智能分类」，点击「导入所选」。

导入后的文件会写入：

```text
assets/uploads/<你填写的文件夹>
```

素材库管理页面的「本地素材」会同步显示这些图片；智能分类结果会写入同名 `.classification.json`。

## 注意

- 只支持网页上的 `http/https` 图片地址，不支持 `blob:` 和 `data:`。
- 局域网使用时，Infinite Canvas 服务需要绑定到局域网可访问的地址，并确保防火墙允许访问对应端口。
- 图钉按钮会打开 Chrome 侧边栏；扫描时会读取当前浏览器窗口正在激活的网页标签页。
- 有些网站会限制防盗链，后端下载失败时插件会显示失败数量。
- 智能分类会调用你选择的平台和模型，请确认 API Key 已在 API 设置中配置。
